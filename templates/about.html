About GeoDora

Explore the World

GeoDora is a comprehensive geography web application that allows you to explore countries, compare statistics, and test your knowledge with interactive quizzes.

Built with Flask and SQLite, this project demonstrates how to create an educational web application with modern UI components and responsive design.
